# Hema's CarSim V0.1
### A Simple Car Simulator based on SFML Library for Udacity C++ NanoDegree Capstone Project.
![screenshot](Screenshot.gif)

#### Dependencies
`sudo apt install libsfml-dev`


#### How To Run:
* `cd build && cmake ..`
* `make`
* `./CarSim`

### How to Drive:
* Q -> Steer Left
* E -> Steer Right
* Arrow Up -> Increase Speed
* Backspace -> Reset
